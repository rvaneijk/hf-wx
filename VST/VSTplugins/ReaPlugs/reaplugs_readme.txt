This installer includes the following VST-compatible plug-ins:

ReaEQ  Unlimited multiband EQ, with selectable filter types and intuitive graphical interface.
ReaComp  Highly configurable compressor, with sidechain, lookahead, knee, customizable detector window and filters, feedback mode. 
ReaXComp  Unlimited multiband compressor, with per-band configurable threshold, knee, attack/release, window size, and a fast and clean interface.
ReaDelay  Unlimited multitap delay, with customizable filters, resolution control, tempo sync, musical or time-based delay, and more.
ReaFIR  Graphical FFT-based EQ and dynamics processing, a flexible tool for surgical EQ/compression, noise gating, and more.
ReaGate  Sidechainable noise gate, with lookahead and hysteresis, and MIDI trigger output for easy hit replacement.
ReaControlMIDI  Flexible MIDI track controls, with full sysex support, fully automatable bank/program select and control change envelopes.
ReaStream  Stream audio and MIDI over the network to other instances of REAPER.
ReaJS  Effect loader for inserting user-scriptable JS audio and MIDI effects in other hosts besides REAPER.

These plug-ins are a subset of the plug-ins that come with REAPER, which you can download from www.reaper.fm.

The REAPER-only versions use 64-bit audio processing end-to-end, whereas these standalone versions use 64-bit processing internally, but 32-bit I/O.
